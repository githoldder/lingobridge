# LingoBridge MVP 预算总表及拆解

> 口径说明：本表面向老师和学院沟通，优先回答“现在能不能公网试用、需要多少钱、后续扩展大概要多少”。  
> 当前部署策略：先复用现有腾讯云服务器完成公网测试；如需正式长期运行，再购买或升级云资源。

---

## 一、预算结论

| 场景 | 首年预算 | 月均 | 适用情况 |
|---|---:|---:|---|
| 当前 MVP 试用版 | 0-300 元 | 0-25 元 | 复用现有腾讯云服务器，仅做公网演示和小范围采集 |
| MVP 正式小范围运行 | 700-1500 元 | 60-125 元 | 购买轻量服务器、对象存储、域名，支持一个班级试用 |
| v1.0 数据采集增强版 | 1500-3000 元 | 125-250 元 | 加入更稳的存储、备份、语音识别资源包 |
| v2.0 智能能力扩展版 | 3000-8000 元 | 250-670 元 | 加实时字幕、翻译、语音评测或更高配置服务器 |

给老师的简版结论：

> 现在可以先用现有腾讯云服务器上线测试，短期几乎不新增成本。若学院希望稳定运行一年，小范围 MVP 预算建议按 1000 元左右准备。后续 AI 评测、实时翻译、会议平台集成属于增强项，不影响当前采集试用。

## 二、当前 MVP 试用版预算

| 项目 | 费用 | 说明 |
|---|---:|---|
| 腾讯云服务器 | 0 元 | 复用已有服务器 |
| 域名 | 0 元 | 先用公网 IP 访问；正式展示可再绑定域名 |
| SSL 证书 | 0 元 | IP 访问阶段可暂不配；域名阶段用 Let's Encrypt |
| 数据库存储 | 0 元 | MVP 暂用服务器本地数据文件 |
| 文件存储 | 0 元 | MVP 暂用服务器本地存储目录 |

风险提示：

| 风险 | 说明 | 建议 |
|---|---|---|
| 本地文件存储丢失 | 服务器重装或误删会影响数据 | 每次采集后备份 |

| 服务器配置不确定 | 并发过高可能卡顿 | 先 10 人试采，再扩容 |

## 四、功能项预算拆解

### 1. 公网访问

| 方案 | 费用 | 优点 | 缺点 |
|---|---:|---|---|
| 公网 IP 访问 | 0 元 | 立刻可用 | 不够正式，不便记忆 |
| 域名 + HTTP | 35-100 元/年 | 好记，便于发给学生 | 仍非 HTTPS |
| 域名 + HTTPS | 35-100 元/年 | 更正式，麦克风/录音兼容更好 | 需要配置证书 |

建议：演示阶段先用公网 IP；组织学生长期采集前配置域名和 HTTPS。

### 2. 服务器

| 配置 | 适用人数 | 预算 |
|---|---:|---:|
| 2核2G | 10-30 人试采 | 300-600 元/年 |
| 2核4G | 30-100 人小范围使用 | 500-1000 元/年 |
| 4核8G | 后续 AI/并发增强 | 1000-2500 元/年 |

说明：当前 MVP 的主要压力来自文件上传、录音/录播存储和前端访问，不建议一开始为大模型购买高配服务器。

### 3. 文件与录音存储

估算：

| 使用量 | 估算 |
|---|---:|
| 1 名学生每天 5 条录音 | 约 1-5MB/天 |
| 30 名学生采集 1 周 | 约 0.2-1GB |
| 100 名学生采集 1 个月 | 约 5-20GB |
| 老师录播 1 小时 | 约 200-800MB |

建议：MVP 可先本地存储；正式运行时启用对象存储或至少做定期备份。

### 4. TTS 中文朗读

| 方案 | 费用 | 适用 |
|---|---:|---|
| 浏览器 TTS | 0 元 | MVP 演示、无预算阶段 |
| Azure/Tencent/阿里云 TTS 免费额度 | 0 元 | 小规模试用 |
| 付费 TTS 资源包 | 100-300 元/年 | 使用量明显增长后 |

建议：当前不把 TTS 作为预算压力点，先保证流程可用。

### 5. ASR/语音识别

| 方案 | 费用 | 适用 |
|---|---:|---|
| 不接入 ASR | 0 元 | MVP 当前阶段 |
| 文件识别资源包 | 90-500 元/年 | 课后批量转写 |
| 实时 ASR | 按分钟计费 | 后续实时字幕或课堂分析 |

建议：先采集录音，不急着上自动识别。等有真实数据后再评估中文识别准确率。

### 6. 翻译/实时字幕

| 方案 | 费用 | 适用 |
|---|---:|---|
| 不接入 | 0 元 | MVP |
| 文本翻译 API | 500-1000 元/年 | 课后翻译 |
| 实时字幕 + 翻译 | 1000-5000 元/年 | v2.0 |

建议：实时翻译对网络、延迟、准确率要求更高，不作为本轮向老师承诺的功能。

## 六、预算说明口径

给老师可以这样说：

> 这个项目当前最大的价值不是昂贵 AI 能力，而是先把中文学习数据采集链路跑通。学生能登录、听中文、录跟读、提交数据，老师能看到结果。短期复用现有腾讯云服务器即可试用，预算接近 0。若学院希望稳定运行一年，建议准备 1000-1500 元，主要用于服务器、域名、存储和备份。AI 语音评分、实时翻译等可以后续根据真实使用情况再追加。

## 七、预算待确认项

| 待确认项 | 影响 |
|---|---|
| 腾讯云服务器当前配置和剩余有效期 | 决定是否需要购买新服务器 |
| 是否需要正式域名 | 决定是否产生域名和 HTTPS 配置工作 |
| 预计参与学生人数 | 决定服务器和存储规格 |
| 是否采集录播视频 | 视频会显著增加存储用量 |
| 是否要接入自动语音识别 | 决定 ASR 预算是否进入当前阶段 |
