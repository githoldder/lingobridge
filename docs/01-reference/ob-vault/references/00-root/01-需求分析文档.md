## 1. 产品概述

### 1.1 产品定位

**老师的课程辅助教学软件** —— 跟着课程走的自学平台

以俄语为媒介（用俄语学中文），与腾讯会议教育版形成合围之势。

### 1.2 核心价值主张

| 价值点 | 说明 |
|--------|------|
| **课后自主学习** | 学生可自由学习，跟随教纲、PPT进行课后复习 |
| **课前预习** | 预习静态电子课件内容 |
| **实时双语字幕** | 类似B站的实时翻译字幕功能 |
| **录播沉淀** | 每次课沉淀成录播视频 |
| **解耦收费** | 可对接任何教育机构，收取国际教育费用 |
| **班级排名** | 积分排名功能，维持用户日活 |

### 1.3 用户痛点（现状）

| 痛点                  | 根因          |
| ------------------- | ----------- |
| 196课时/4课时，一周上一次，间隔久 | 缺乏课后自主学习工具  |
| 每次一个PPT，无课后复用       | 缺少数字化课程资产沉淀 |
| 学生预习只能看静态课件         | 无互动式预习系统    |
| 课后只能用打印教材测试         | 无数字化测评反馈    |
| 腾讯会议无法实时翻译          | 缺乏双语字幕能力    |

## 2. 用户与角色

### 2.1 用户体系

| 角色      | 职责              | 备注        |
| ------- | --------------- | --------- |
| **老师**  | 发布课程、录制PPT、管理学生 | Web端      |
| **学生**  | 课前预习、课后练习、发音测试  | Web端（自适应） |
| **管理员** | 平台管理、机构入驻、学校认证  | Web端      |

### 2.2 用户场景

**学生端：**
- 课后练习发音和音标
- 韵脚、声母练习
- 课前预习
- 当日课程作业、课程练习
- 班级积分排名（可见，仅限班级内排名）

**老师端：**
- 上传课件/PPT/视频
- 发布课程
- 批改/查看学生数据
- 实时翻译上课

### 2.3 核心优势（新增）

| 优势 | 说明 |
|------|------|
| **师资力量** | 老师是对外交流使者 |
| **资源优势** | 学校有哈萨克斯坦留学生资源 |
| **机构背景** | c2k国际教育机构成员 |
| **教学经验** | 已跑通线上线下结合的疫情时期"网课直播教学"模式 |

### 2.4 目标用户

- **主要用户**：哈萨克斯坦留学生
- **次要用户**：俄语学习者（成人/青少年）

## 附录：调研结论

**市场机会：**
- 俄罗斯在线教育市场2021年达533亿卢布（CAGR 20%）
- 俄语培训市场规模246亿卢布
- 哈萨克斯坦海外留学生91,860人，其中11,764人在中国学习
- 全球在线语言学习市场2026年达215亿美元（CAGR 16.74%）
- "一带一路"政策驱动俄语人才需求（52个非洲国家已签署）

**政策风险：**
- 在线教育监管政策（VIPKID等合规参考）
- 外教资质要求
- 跨境支付合规

**技术可行性：**
- 语音识别、机器翻译技术成熟
- 视频处理、CDN分发方案标准化

**竞品调研补充：**
- 俄语学习：SpeakPal（免费）、欧那俄语（13年）、灵犀俄语、鹅语菌
- 中文学习Duolingo（全覆盖）、HelloChinese（发音）、Yoyo Chinese（1000+课程）
- 市场空白：俄语→中文垂直赛道 + 教学辅助 + 课程跟随
