## 3. Sprint规划

> **Sprint周期**：1周（周五→下周四），单人Scrum
> **工作模式**：Vibe Coding（AI辅助），每日可用开发时间约 4-6h（考研并行）

### 方案A：IP+端口直接访问（最快上线）

> 甲方提供香港云服务器后，用 `http://IP:3000` 直接访问，无需域名、无需备案、无需Nginx。

| Sprint | 周期 | 目标 | 交付物 |
|--------|------|------|--------|
| **S0** | 第1周 | **基础设施 + 前端骨架** | ① 阿里云轻量HK开箱（甲方出钱/转交凭证）<br>② SSH连通 + Docker安装<br>③ Next.js项目脚手架 + Layout三语切换<br>④ 学生端课程列表页（Mock数据） |
| **S1** | 第2周 | **核心闭环：PPT浏览 + TTS** | ① PPT/PDF上传 + 解析为图片页<br>② 课程页翻页浏览<br>③ TTS按钮接入腾讯云API<br>④ 录音采集（MediaRecorder） |
| **S2** | 第3周 | **录音管理 + 老师端** | ① 录音上传/播放/删除/下载<br>② 老师端：课件上传 + 课程管理<br>③ 录屏功能（getDisplayMedia）<br>④ PostgreSQL数据持久化 |
| **S3** | 第4周 | **联调 + 部署上线** | ① Docker Compose一键拉起<br>② 服务部署到HK服务器<br>③ 端到端测试（TTS→录音→回放）<br>④ **上线测试：`http://IP:3000`** |

**方案A结论：4个Sprint（4周）后可上线测试**

---

### 方案B：域名 + 国内可访问（稳定正式）

> 需额外处理域名注册、Nginx反向代理、SSL证书。ICP备案可不做（HK服务器）。

| Sprint | 周期 | 目标 | 交付物 |
|--------|------|------|--------|
| S0-S3 | 第1-4周 | 同方案A | 同方案A |
| **S4** | 第5周 | **域名 + Nginx + HTTPS** | ① 域名注册（.com，约70元/年）<br>② Nginx反向代理配置<br>③ Let's Encrypt SSL证书<br>④ DNS解析到HK服务器IP<br>⑤ **正式访问：`https://lingobridge.com`** |

**方案B结论：5个Sprint（5周）后可正式上线**

---

## 4. 两种上线方案对比

| 维度 | 方案A：IP+端口 | 方案B：域名+HTTPS |
|------|---------------|-----------------|
| **上线时间** | 4周 | 5周 |
| **访问方式** | `http://43.xxx.xxx:3000` | `https://lingobridge.com` |
| **费用** | 服务器288元/年 | 服务器288 + 域名70 = 358元/年 |
| **适用场景** | 内部测试、甲方演示 | 对外正式使用、学生日常访问 |
| **HTTPS** | ❌ 无（浏览器可能拦截录音权限） | ✅ 有 |
| **录音功能** | ⚠️ 浏览器安全策略可能阻止非HTTPS页面的录音权限 | ✅ 正常工作 |
| **ICP备案** | 不需要 | 不需要（HK服务器） |
| **国内访问** | ✅ 可访问（HK服务器CN2回程） | ✅ 可访问 |
| **国际访问** | ✅ 可访问 | ✅ 可访问 |
| **专业度** | 低（IP地址不专业） | 高（正式域名） |

> ⚠️ **关键提醒**：浏览器 `getUserMedia`（录音/摄像头）在非HTTPS环境下**多数浏览器会拒绝或警告**。方案A如果用HTTP+IP，录音功能大概率不可用。建议至少方案B，或方案A临时用自签证书。

### 推荐策略：A→B 渐进式

1. **先跑方案A**（4周）：用 `http://IP:3000` 验证功能闭环，内部测试
2. **第5周切方案B**：加域名+Nginx+HTTPS，解决录音权限问题，正式交付甲方

## 9. 甲方沟通要点

1. **服务器必须先到位**，否则无法开发部署。建议本周内完成购买。
2. **IP访问 vs 域名访问的区别**：
   - IP:端口 = 能用但不专业，录音功能可能有浏览器安全限制
   - 域名+HTTPS = 正式可用，录音功能正常，推荐
   - 两者差1周开发时间 + 70元/年域名费
3. **MVP只做学习+录音**，不做实时翻译、不做弹幕、不做AI打分。这些是v2.0的事。
4. **预算约700元/年**（不含域名），详见预算表。
5. **4周可内部测试，5周可正式上线**（前提：服务器本周到位）。
