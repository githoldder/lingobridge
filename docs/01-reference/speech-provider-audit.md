# Speech Provider Audit — 官方额度与控制台实测

> **项目**: LingoBridge 中文教学平台
> **审计目的**: 复核 TTS/ASR 提供商的免费额度、认证门槛、并发限制和商用限制，为上台演示和产品集成提供真实依据。
> **审计日期**: 2026-05-21
> **审计范围**: 火山引擎/豆包语音、百度智能云语音、ElevenLabs、Azure Speech Services
> **状态**: 初始审计 — 额度以官方文档和控制台实测为准，定期复核

---

## 审计准则

1. 额度数据来源：官方定价页 / 控制台 / 认证邮件，不以第三方博客、社区帖或 AI 生成为准。
2. 不出现"永久免费、不限流、不会封 IP"等绝对承诺，均标注风险等级。
3. 上台演示优先使用预生成 MP3，降低实时 API 依赖风险。
4. 每个提供商标注认证状态和实测结果。

### 1.1 基本信息

| 项目 | 内容 |
|---|---|
| 官网 | https://www.volcengine.com/product/voice |
| 控制台 | https://console.volcengine.com/speech |
| API 文档 | https://www.volcengine.com/docs/6301 |
| 服务类型 | TTS（语音合成）、流式 ASR（实时语音识别） |

### 1.2 免费额度与认证

| 项目 | 额度 | 来源 |
|---|---|---|
| TTS 试用字符 | 约 **20,000 字符/半年** | 豆包语音控制台 — 资源包管理 |
| 流式 ASR 试用时长 | 约 **20 小时/半年** | 豆包语音控制台 — 资源包管理 |
| 认证门槛 | 实名认证（个人/企业）；需绑定火山引擎账号 |
| 开通方式 | 控制台创建应用 → 获取 AppID + AccessToken |

> **风险标注**: "20,000 字符/半年" 是豆包语音标准试用包。常见错误表述如"100 万字符/月"或"500 小时/月"均与此不符，需以控制台实际显示为准。试用额度不可续期，用尽后调用返回 403。

### 1.3 并发与 QPS 限制

| 限制项 | 数值 | 备注 |
|---|---|---|
| TTS QPS | 免费试用 **20 QPS** | 企业认证后可申请提升 |
| ASR QPS | 免费试用 **10 QPS** | 同上 |
| 并发连接 | 无硬性上限，受 QPS 约束 |

### 1.4 商用限制

- 免费资源包不可用于商业发布，超出试用额度需购买付费资源包。
- 付费 TTS 约 ¥2.5/万字符（标准），¥8/万字符（精品）。
- 生成音频可缓存，但不得转售或重新分发 API 服务本身。
- 中国区域数据保留在火山引擎国内节点。

### 2.2 免费额度与认证

| 项目 | 额度 | 来源 |
|---|---|---|

| 认证有效期 | Access Token 有效期 30 天，需定时刷新 |

### 2.4 商用限制

- 免费额度仅限测试，商用需购买付费资源包。
- 付费 TTS 约 ¥4/万字符（短文本），¥6/万字符（长文本）。
- 百度语音服务受百度智能云服务条款约束，不得用于非法或高敏感场景。
- 音频缓存一般允许但需遵守百度 AI 服务协议。

## 6. 风险清单

| 风险 | 影响 | 应对 |
|---|---|---|
| 免费额度用尽 | 实时 TTS 返回 403/429 | 使用预生成 MP3；缓存回退 |
| 俄语 TTS 质量不达标 | 演示效果差 | 提前用 ElevenLabs 生成并试听 |
| 国内网络无法访问 ElevenLabs | 俄语生成失败 | 使用预生成；Azure 俄语备选 |
| 实名认证未通过 | 无法使用国内 provider | 提前完成认证；海外 provider 备选 |
| 并发触发限流 | 延迟增加 | 串行调用；预生成 |
| 控制台额度与文档不符 | 额度评估不准 | 以控制台实际显示为准 |

---

## 7. 演示策略

1. **上台演示 TTS 全部使用预生成 MP3**，不实时调用任何云 API。
2. 预生成使用 `scripts/demo/generate-tts-assets.ts` 脚本，生成结果写入 `public/demo-audio/`。
3. 保留 browser Fallback（浏览器内置 TTS）作为极端断网兜底，但音质不承诺演示级。
4. 演示前检查清单：音频文件完整 → 音量正常 → 断网测试通过。
