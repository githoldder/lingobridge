# LingoBridge 人工智能课程大作业报告

本目录是课程大作业 LaTeX 报告工程。

## 题目

中文题目：基于 DKT 知识追踪与 Levenshtein 距离校准的国际中文自适应学习推荐系统

英文题目：An Adaptive International Chinese Learning Recommendation System Based on Deep Knowledge Tracing and Levenshtein Distance Calibration

## 构建

```bash
./build.sh
```

构建产物输出到：

```text
output/报告.pdf
```

## 说明

报告围绕 LingoBridge 的国际中文学习场景，将 DKT 知识追踪用于学生掌握状态预测，将 Levenshtein 距离用于拼音/字幕校准，并结合知识点依赖关系生成自适应练习推荐。
