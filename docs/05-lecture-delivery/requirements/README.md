# 人工智能课程大作业交付要求

来源文件：

- `人工智能课程大作业指导书(2025-2026-1).pdf`

读取时间：2026-06-02

## 课程基本信息

- 课程名称：人工智能
- 课程代码：5COM2003
- 适用专业：软件工程
- 大作业占总成绩：50%

## 硬性任务要求

指导书要求大作业围绕人工智能问题求解、数据分析和工程实现展开。对 LingoBridge v2.0 AI 模块而言，必须落实为以下可交付物：

1. 选择并说明课程设计题目，明确设计指标和功能要求。
2. 收集资料，完成背景、现状综述和总体方案论证。
3. 完成数据预处理，观察数据特征并处理为方便建模的格式。
4. 完成功能/函数设计，明确模块划分。
5. 完成算法设计，基于选题实现数据挖掘、机器学习或深度学习分析。
6. 完成交互设计，体现教师端、学生端和管理端的 AI 模块展示。
7. 原则上采用 Python 编码，实现全部核心数据处理和算法功能。
8. 完成运行与测试，设计测试用例并输出结果。
9. 完成大作业报告，报告需说明小组分工。
10. 提交报告文本、Python 代码与输出结果。

## 考核权重

- 沟通交流：20%
- 代码实现：30%
- 答辩演示：20%
- 文档报告：30%

## 与本项目的对齐解释

LingoBridge 本身已经是较成熟的教学平台，课程大作业不能只把平台包装成软件工程项目。新增 AI 模块必须形成独立的人工智能闭环：

- 有可导出的学习行为数据。
- 有 Python 数据处理脚本。
- 有明确算法与模型训练/评估过程。
- 有推荐或预测结果。
- 有三端界面承载分析结果。
- 有报告中的实验、图表、指标和反思。

因此，本项目应从指导书参考选题中的“自然语言处理/互联网/人文社科/数据分析”方向扩展为面向国际中文教育的综合应用开发题，而不是复用人脸识别、房价预测等与平台无关的题目。
