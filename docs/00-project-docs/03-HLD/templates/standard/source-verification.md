# 03-HLD 标准图来源核验清单

## 当前状态

- 已移除来源不明的 Class / Sequence 视觉 benchmark。
- 原 benchmark 无 OMG 章节号、无页码、无来源 URL，且画风与文件元数据均不符合“标准截图”要求。

## 推荐核验源

| 图件类型 | 推荐来源 | 可信层级 | 截图/搬运要求 |
| --- | --- | --- | --- |
| 类图、组件图、时序图、部署图 | OMG UML 2.5.1 Specification | 官方规范 | 记录章节号、页码、官方 PDF URL、访问日期 |
| 软件设计文档结构 | GB/T 8567-2006《计算机软件文档编制规范》 | 国家标准 | 记录标准号、页码、URL/馆藏来源 |
| 软件设计描述 | IEEE 1016-2009 或课程指定材料 | 标准/教材 | 记录标准名称、章节号、页码 |

## 已知可信线索

- OMG UML 2.5.1 官方入口：`https://www.omg.org/spec/uml/`
- 国家标准公开系统可用于核验标准题名、状态与编号；全文截图应走学校数据库、官方授权渠道或用户手动截图。

## 禁止事项

- 禁止使用无来源图片作为 UML 标准对照。
- 禁止把 PlantUML 渲染图、AI 生成图、博客示意图标注为 OMG/GB 原图。
- 禁止在 HLD 图中画出与代码事实不一致的模块、接口或数据库对象。
