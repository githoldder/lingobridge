## 1. 产品简介
LingoBridge 是一个极简的在线对外中文学习工具。老师可以轻松上传 PDF 课件并导入题库进行直播授课；学生无须下载客户端，直接在浏览器端加入课堂，观看课件并完成老师留下的跟读任务。

### 1.1 系统与环境要求
- **终端**：支持 Windows / macOS / 平板。
- **浏览器推荐**：最新版的 Google Chrome 或 Safari（确保授予麦克风权限以支持录音）。

## 2. 教师端操作指南

### 2.1 课程与班级管理
1. 登录教师账号后，在首页看到“我的课程”。
2. 点击已有课程卡片，进入【课程编辑页】。
3. 此页面可统一管理该课程下的所有：学生、上传课件、作业记录和每一次的 Live。

### 2.2 上传课件与布置作业
- **课件上传**：点击“上传课件”，支持导入 PDF 或 PPTX。文件上传后系统会解析并在上课时作为背景展示。
- **作业导入**：提供标准的 Excel 模板，教师可一次性导入拼音、词汇、俄语/哈萨克语翻译，系统自动生成跟读任务节点。

### 2.3 直播课堂与画笔
1. 针对一个班级创建“新 Live”。
2. 进入 Live 界面后，左侧显示已准备的课件，右侧可查阅学生名单。
3. 底部提供“画板”功能，开启后可用清晰圆珠笔线条在课件上做标注。
   *注：关闭画板模式时已有的笔迹仍将保留；如果需要清除屏幕，请点击“清除笔迹”按钮。*

## 3. 学生端操作指南

### 3.1 访客门禁
- 您可以无需登录直接访问网站并查看公开的课程大纲。
- 当需要加入具体班级、上交作业时，系统会弹出登录提示，请使用管理员分配的学生账号登录。

### 3.2 加入课堂与录音
1. 老师开启直播后，进入对应课程，点击“进入教室”。
2. 您的屏幕将同步老师的课件展示内容。老师并未开放全班画板权限时，您无法涂鸦。
3. 课后，点击当堂课对应的“作业节点”，允许浏览器调用麦克风后进行跟读练习，点击“提交”后成绩将自动记录。

## 4. 管理端 (Admin) 操作说明
管理员通过浏览器直接访问 `/admin` 路径并使用超级账号登录：
- **账号管理**：可重置师生密码或封禁违规账号。
- **资源监控**：可纵览系统所有的 PDF/Excel 文件以及 `db.json` 数据快照状态。
