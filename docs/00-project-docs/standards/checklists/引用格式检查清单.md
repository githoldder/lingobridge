# 引用格式检查清单

> 本清单用于检查文档中参考文献引用格式是否符合 GB/T 7714—2015《信息与文献 参考文献著录规则》。适用于所有学术文档、工程文档中的引用部分。

---

## A. 引用格式自检

### A.1 期刊文章
- [ ] 作者姓名格式正确（姓前名后，如"张三"而非"Zhang San"）
- [ ] 刊名完整，不使用缩写
- [ ] 有卷号和期号
- [ ] 有页码范围
- [ ] 无 DOI 时标注访问日期和 URL（有 DOI 时可不标）

### A.2 图书
- [ ] 有出版地（城市名）
- [ ] 有出版者
- [ ] 有出版年
- [ ] 译者信息格式正确（如有）
- [ ] 版本信息正确（如"第3版"）

### A.3 学位论文
- [ ] 有学位授予单位
- [ ] 有城市
- [ ] 有年份

### A.4 国家标准
- [ ] 标准号正确（GB/T xxx-xxxx）
- [ ] 有发布日期（年月日）
- [ ] 有标准全称

### A.5 国际标准
- [ ] 有发布机构缩写
- [ ] 有标准号
- [ ] 有年份

### A.6 网页资源
- [ ] 有更新日期或发布日期
- [ ] 有访问日期
- [ ] 有完整 URL
- [ ] 有作者或机构（匿名不可用）

---

## B. 引用一致性自检

- [ ] 正文中每个引用编号在参考文献中都有对应条目
- [ ] 参考文献中每条在正文中都有引用
- [ ] 正文引用编号与文后编号顺序一致（或按姓氏排序，以学校要求为准）
- [ ] 同一文献在全文中引用格式一致
- [ ] 作者姓名拼写一致（同一作者在全文献中写法统一）

---

## C. 引用质量自检

- [ ] 引用来源可信（L1-L3，无 L4/L5）
- [ ] 引用的是一手文献（非转引，除非无法获取原始文献）
- [ ] 无 AI 生成内容作为引用来源
- [ ] 引用数量充足（论文通常≥ 30篇）
- [ ] 近年文献占比合理（通常≥ 40%，技术文档可放宽）
- [ ] 核心观点有原始文献支撑（非道听途说）

---

## D. 常见错误警示

| 错误类型 | 正确做法 |
|---|---|
| 期刊名写缩写 | 写全称，如"计算机学报"而非"CJ" |
| 作者姓名写反 | "张三"而非"Zhang San"（GB/T 7714规定中文姓前名后） |
| 缺少页码 | 期刊文章必须标注起止页码 |
| DOI 和 URL 同时写 | 有 DOI 时可不写 URL |
| 网页无作者 | 找机构作者，或不使用该来源 |
| 正文章节引用用上标 | 按学校/期刊要求（GB/T 7714 允许顺序编码制或著者-出版年制） |
| 电子资源无访问日期 | 网页必须标注访问日期 |
| 中文文献用英文格式 | 中文文献用中文格式，英文文献用英文格式 |

---

## E. GB/T 7714—2015 著录格式速查

```
期刊：[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 页码.

图书：[序号] 作者. 书名[M]. 出版地: 出版者, 年.

学位论文：[序号] 作者. 题名[D]. 城市: 学位授予单位, 年.

国家标准：[序号] 发布机构. 标准编号 标准名称[S]. 发布日期.

网页：[序号] 作者. 题名[EB/OL]. (日期)[日期]. URL.
```

---

## F. 工具推荐

- **Zotero**：免费，支持 GB/T 7714 输出格式
- **EndNote**：付费，高校通常有授权
- **Mendeley**：免费
- **NoteExpress**：中文友好
- **LaTeX + natbib/bibtex**：自动化管理，格式自动生成

> 注意：工具生成的引用需人工核对正确性，工具输出≠100% 正确。
