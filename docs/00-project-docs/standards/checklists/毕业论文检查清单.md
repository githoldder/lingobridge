# 毕业论文检查清单

> 本清单用于毕业论文提交前的全面审查。依据 GB/T 7713.1—2025 和高校本科毕业设计规范。

---

## A. 形式审查

- [ ] 论文篇幅符合学校要求（通常≥ 10000字）
- [ ] 封面格式符合学校模板
- [ ] 摘要（中英文）符合规范，无图表，无参考文献
- [ ] Abstract 与中文摘要内容一致
- [ ] 关键词3-5个，中英文对应
- [ ] 目录页码正确，与正文对应
- [ ] 正文字体、字号、行距符合模板
- [ ] 页眉页脚格式符合要求
- [ ] 图、表、公式编号连续
- [ ] 所有图表有题注
- [ ] 无错别字、标点错误
- [ ] 查重率符合学校要求（通常≤ 30%）

---

## B. 结构审查

- [ ] 第1章 绪论：说明"为什么做"
- [ ] 第2章 相关技术与理论基础：说明"用什么做"
- [ ] 第3章 [核心章节]："怎么做"
- [ ] 第4章 [核心章节]："做出了什么"
- [ ] 第5章 总结与展望：总结成果，指出不足
- [ ] 无缺章节（对照学校模板检查）
- [ ] 各章篇幅分布合理（核心章节占比最大）

---

## C. 内容审查

- [ ] 研究目的明确
- [ ] 国内外研究现状有文献综述（非简单罗列）
- [ ] 核心章节有实质内容（不是凑字数）
- [ ] 创新点明确（有别于现有研究/系统的独特贡献）
- [ ] 研究方法科学合理
- [ ] 实验/开发/设计工作量充足
- [ ] 结论有数据/实验支撑

---

## D. 标准符合性

- [ ] 符合 GB/T 7713.1—2025 结构要求
- [ ] 参考文献符合 GB/T 7714—2015 格式
- [ ] 量和单位使用 GB/T 3100 系列
- [ ] 标点符号使用 GB/T 15834—2011
- [ ] 图表格式规范（三线表、无多余竖线）
- [ ] 引用格式正确（正文有标注，文后有对应）

---

## E. 引用与参考文献

- [ ] 引用文献量≥30篇
- [ ] 近5年文献≥40%
- [ ] 参考文献格式统一（逐条核对）
- [ ] 正文中每个编号都有对应文献
- [ ] 无引用未在文后列出的文献
- [ ] 无虚假引用（确有其文献）
- [ ] 无 AI 生成内容作为事实来源

---

## F. 答辩准备

- [ ] PPT 内容精炼（每页1个核心信息）
- [ ] PPT 结构与论文一致
- [ ] 能回答"为什么做、做了什么、怎么做的、做得如何"
- [ ] 能清晰阐述创新点
- [ ] 对参考文献内容熟悉
- [ ] 对技术细节掌握到位
- [ ] 控制在答辩时间要求内

---

## G. 提交前最终确认

- [ ] 全部作者信息填写完整（姓名、学号、专业、指导教师）
- [ ] 独创性声明已签字
- [ ] 导师签字（或电子签）已获取
- [ ] PDF 版本已生成，格式无误
- [ ] 纸质版打印无误
- [ ] 提交清单已核对（论文 + 光盘/电子版 + 相关材料）
