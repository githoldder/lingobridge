# Domain Coverage Summary

Verified date: 2026-06-02

This summary follows the four-level source hierarchy:

- L1 `normative-baseline`: standards, formal specifications and official standard metadata.
- L2 `official-guidance`: official guidance from standards bodies, professional associations, institutions or large design systems.
- L3 `case-gallery`: examples, templates, code and good/bad visual cases.
- L4 `discovery-index`: curated indexes for discovering more sources only.

## Enforcement Notes

1. L4 resources are discovery indexes only and cannot be used for compliance decisions.
2. L3 resources are examples and few-shot references only.
3. L2 resources may provide design rules, but cannot override L1 definitions.
4. Only L1 resources can justify “conforms to standard” or “violates standard.”
5. PDF resources cannot enter `case-figs` as screenshots until page number, figure number and figure title are recorded.
