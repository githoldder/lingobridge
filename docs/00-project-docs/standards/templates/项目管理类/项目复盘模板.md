## 填写说明

1. 复盘应在项目结束后1周内完成
2. 所有参与核心工作的成员都应参与复盘
3. 讨论时对事不对人，聚焦改进而非追责
4. 改进措施须明确负责人和截止日期
5. 复盘结果应在团队内共享
