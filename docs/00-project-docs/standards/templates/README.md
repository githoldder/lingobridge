## 使用说明

1. **直接复制模板内容**，在对应位置填写实际内容
2. 每个模板包含：文档目的、适用场景、目标读者、参考标准、文档结构、填写说明、检查清单
3. **不要只写框架**——模板正文部分是可直接提交的质量内容
4. 如需调整模板结构，可基于本 Skill 的 `09_文档质量审查清单.md` 进行质量检查

---

## 模板质量标准

每个模板都应满足：

- ✅ 有完整的 Markdown 正文（可直接复制使用）
- ✅ 包含所有必要章节
- ✅ 有填写说明（灰色斜体或注释说明）
- ✅ 有检查清单（防止遗漏）
- ✅ 符合对应国家标准的章节要求
- ✅ 有常用格式的示例内容

---

## 如何创建新模板

如需为本 Skill 添加新模板：

1. 在对应分类目录下创建 `.md` 文件
2. 文件名格式：`{文档类型}模板.md`
3. 包含以下标准结构：
   ```markdown
   # {文档类型}模板
   
   ## 文档元数据
   ## 模板正文
   ## 填写说明
   ## 检查清单
   ## 常见错误
   ```
4. 更新本 README.md 的索引表
   ```
