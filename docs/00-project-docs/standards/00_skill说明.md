## 规范依据

本 Skill 构建参考以下标准：

**国家标准（必依）：**
- GB/T 7713.1—2025《信息与文献 编写规则 第1部分：学位论文》
- GB/T 7713.2—2022《学术论文编写规则》
- GB/T 7714—2015《信息与文献 参考文献著录规则》
- GB/T 8567—2006《计算机软件文档编制规范》
- GB/T 9385—2008《计算机软件需求规格说明规范》
- GB/T 15834—2011《标点符号用法》
- GB/T 1.1—2020《标准化工作导则》

**国际标准（参考）：**
- ISO/IEC/IEEE 12207:2017 软件生命周期过程
- ISO/IEC/IEEE 29148:2018 需求工程
- ISO/IEC/IEEE 15288:2015 系统生命周期过程

**行业实践（参考）：**
- IEEE 830 软件需求规格说明
- IEEE 1016 软件设计描述
- IEEE 829 软件测试文档
