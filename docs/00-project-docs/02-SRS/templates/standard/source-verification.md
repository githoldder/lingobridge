# 02-SRS 标准图来源核验清单

## 当前状态

- 已移除来源不明的 DFD / Use Case 视觉 benchmark。
- 原有 benchmark 存在文件扩展名与真实格式不一致、无来源、无页码、疑似 AI 生成的问题，不可作为交付依据。

## 推荐核验源

| 图件类型 | 推荐来源 | 可信层级 | 截图/搬运要求 |
| --- | --- | --- | --- |
| SRS 文档结构与需求说明 | 国家标准公开系统：GB/T 9385-2008《计算机软件需求规格说明规范》 | 官方标准页面 | 记录标准号、官方页面 URL、访问日期；如需全文截图，优先使用学校数据库或合法授权 PDF |
| 数据流图符号 | GB/T 1526-1989 | 国家标准 | 截图记录页码，图中不得出现自造英文标题或营销装饰背景 |
| 用例图、状态图 | OMG UML 2.5.1 Specification | 官方规范 | 记录 UML 章节、页码、官方 PDF URL |
| 需求图教学示例 | 学校指定软件工程教材 | 教材 | 记录书名、版次、页码；只作课堂对照，不冒充国标原图 |

## 已知可信线索

- 国家标准公开系统 GB/T 9385-2008 页面：`https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=2790825C43AD0B69E3C38C140BFFCFE6`
- OMG UML 2.5.1 官方入口：`https://www.omg.org/spec/uml/`

## 禁止事项

- 禁止保留没有来源说明的 `standard_illustrations`。
- 禁止使用 AI 生成 DFD、UML 图作为“标准对照图”。
- 禁止只写“GB/T 8567”却不说明具体页码和图形符号来源。
