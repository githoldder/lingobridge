# 01-FA 标准图来源核验清单

## 当前状态

- 已移除来源不明的 AI 风格标准对照图。
- 本阶段暂不存放标准截图，避免把不可核验图片误当成国标样例。

## 推荐核验源

| 用途 | 推荐来源 | 可信层级 | 处理要求 |
| --- | --- | --- | --- |
| 流程图、数据流相关符号 | GB/T 1526-1989《信息处理 数据流程图、程序流程图、系统流程图、程序网络图和系统资源图的文件编制符号及约定》 | 国家标准，优先找官方或馆藏来源 | 截图必须记录标准号、页码、URL/馆藏来源、访问日期 |
| 软件文档结构 | GB/T 8567-2006《计算机软件文档编制规范》 | 国家标准 | 只用于文档结构和图件归属核验，不伪称其给出所有 UML 画法 |
| UML/建模符号 | OMG UML 2.5.1 Specification | 官方规范 | 截图必须来自 OMG 官方 PDF，记录章节号和页码 |

## 禁止事项

- 禁止使用 AI 生成图片冒充“国标/行标标准图”。
- 禁止使用无来源、无页码、无 URL 的截图。
- 禁止把第三方博客二次整理图直接搬入仓库作为权威标准。
