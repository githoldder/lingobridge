# MVP Implementation Workflow

1. Read `context/project-brief.md`.
2. Read `prds/prd.md`.
3. Pick one task from `prds/prd.json`.
4. Implement only that task's scope.
5. Run relevant checks.
6. Update task notes or produce a concise handoff.
7. Audit against `.agent/rules/mvp-scope.md` and `.agent/rules/security.md`.

## Recommended Order

1. Backend health/auth.
2. Courseware upload and course pages.
3. Student TTS/recording APIs.
4. Teacher replay APIs.
5. Frontend API client/login.
6. Teacher upload UI.
7. Student practice UI.
8. Teacher media lifecycle and replay.
9. Schedule replay history.
10. Tencent Cloud public test.
