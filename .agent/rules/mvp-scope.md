# MVP Scope Rules

## Source Of Truth

The approved source PRD for repository execution is maintained in:

- `prds/prd.md`
- `prds/prd.json`

Historical private notebook paths must not be used as source links in prompts,
deliverables, citations, or handoff reports. If a private notebook draft was
used to seed the repository copy, treat the repository copy above as the
auditable baseline for all agent work.

## Must Stay In Scope

- PPT/PDF courseware upload and course page generation.
- Excel exercise import linked to course/page.
- Chinese TTS.
- Student recording upload and management.
- Teacher replay recording and student replay history.
- Camera/mic/screen lifecycle bugfix.
- Tencent Cloud public test.

## Must Stay Out Of Scope

- Online PPT editing.
- Real-time bilingual subtitles.
- Bullet comments/social features.
- Multi-tenant SaaS implementation.
- International payment.
- AI pronunciation scoring.
