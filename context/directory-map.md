# Directory Map

This repository uses a light AI-friendly structure while keeping the existing Vite frontend at the root for now.

| Path | Purpose |
|---|---|
| `.agent/` | AI collaboration rules, skills, workflows |
| `analysis/` | Business analysis, process simulation, scope analysis |
| `context/` | Short, important project context |
| `docs/` | Reference docs and technical notes |
| `drafts/` | Unapproved drafts and archive |
| `prds/` | Approved PRD and machine-readable task packages |
| `prompts/` | Worker prompts for other AI agents |
| `templates/` | Reusable templates |
| `backend/` | Reserved for MVP backend |
| `tests/` | E2E, fixtures, acceptance artifacts |
| `docker/` | Nginx/compose deployment assets |
| `scripts/` | Deploy, seed, health-check scripts |
| `src/` | Current Vite frontend source |
| `public/` | Current Vite public assets |
